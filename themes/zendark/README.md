# Zendark GTK theme

A dark GTK theme inspired by the [Zenburn](https://github.com/jnurmine/Zenburn) vim color scheme, originally created for use with i3 window manager.
It was designed to reduce eye strain during long coding sessions while maintaining aesthetic appeal.

**This theme is currently in maintenance mode. **

I'm quite satisfied with the current Adwaita Dark theme, so there probably won't by any updated in near future.
